오후 : CSS code 작성 방법
        - 요소의 실제 크기 (width/height : margin + padding + border)
          : border
          : outline
        - word-break: normal; (어절 기준 줄바꿈), 
          word-break: break-all; (영어),
          word-break: keep-all; (한글)
        - box-sizing
           : content-box (기본값 width + margin + padding + border)
           : border-box (width(padding+border) + margin)
        - 넘치는 컨텐츠 제어 overflow
           : inherit (상속은 현재 부모나 조상 요소에 overflow가 적용된 게 없으면 성격을 물려받지 못함)
           : visible (기본값)
           : hidden (숨김)
           : auto (자동)
           : scroll (스크롤)
           : overflow-x,y
        -  white-space: (wrap; / nowrap;)
